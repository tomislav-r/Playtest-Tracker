# Playtest-Tracker

Projekt se može pokrenuti iz IntelliJ IDEA okruženja.

Koraci:

Preuzeti projekt.
Otvoriti projekt u IntelliJ IDEA.
Kreirati MySQL bazu playtest_tracker.
Provjeriti podatke za spajanje na bazu u application.properties.
Pokrenuti Spring Boot aplikaciju.

Nakon pokretanja aplikacija je dostupna na adresi:

http://localhost:8080
